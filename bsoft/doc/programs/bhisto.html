Bsoft program

bhisto

Usage: bhisto [options] input.img output.img

Calculates the histogram of an image and associated statistics.

Actions:

-bins 256Calculate histogram over a number of bins.
-countsCalculate a counts histogram and adjust the image.
-fit gauss,2Do a fit to the histogram (gaussian, poisson, counts).
Second argument indicates number of gaussians.
Counts option will convert the image to discrete counts.
-otsuCalculate the Otsu inter-set variance.
-multi 5Calculate multiple thresholds for the given number of clusters.
-percentilesCalculate the percentiles for an image.

Parameters:

-verbose 7Verbosity of output.
-datatype uForce writing of a new data type.
-select 1Select an image (default all).
-sampling 2,3.5,1Sampling (angstrom/voxel, a single value sets all three).

Input:

-Histogram histo.psPostscript input file name (instead of image).

Output:

-Postscript histo.psPostscript output file name.
