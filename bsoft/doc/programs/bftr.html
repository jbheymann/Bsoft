Bsoft program

bftr

Usage: bftr [options] file.spi file.mrc

Resizing an image using Fourier transformation.

Actions:

-scale 1.56Isotropic scale (required to resize the data).
-translate 0.3,-4.1,3.3Translation.
-filterextremesRemove extreme regions (such as micrograph labels)

Parameters:

-verbose 7Verbosity of output
-datatype uForce writing of a new data type
-origin 0,-10,30Set origin, used with -size option (default 0,0,0)
-size 22,33,14New size (voxels, default from data)
-sampling 2,3.5,1Sampling before rescaling (angstrom/voxel, a single value sets all three)
-resolution 25.3,200Resolution limits (angstrom).

Input:

-reference ref.mapReference for resizing.
