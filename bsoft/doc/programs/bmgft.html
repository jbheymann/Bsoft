bmgft

Usage: bmgft [options] input.star [input2.star]

Fourier transforms micrographs from a tomographic tilt series.

Actions:

-removemarkers 14Mask out markers with this radius (pixels) before transformation.

Parameters:

-verbose 7Verbosity of output.
-datatype uForce writing of a new data type.
-select 23,45,66Micrograph selection (default all).
-size 1024,1024,200Intended reconstruction size (default from micrograph).
-scale 0.5Scale of intended reconstruction (default 1).
-pad 3Image padding factor (default 2).
-fill 127Fill value for erasing/painting markers (default average).

Output:

-output file.starOutput STAR file name.
