bcatmol

Usage: bcatmol [options] input.pdb [input2.pdb ...]

Concatenates sets of coordinates with shifts from sets of files.

Actions:

-select CAAtom selection (default all).
-rename DRename molecules from the given letter.

Parameters:

-verbose 7Verbosity of output.
-box 10,10,10Size of enclosing box (angstrom, default from coordinates).
-pbcResolve periodic boundary bonds before catenation.

-parameters parm.starAtomic properties parameter file (default atom_prop.star).

-output output.pdbOutput coordinate file (default temp.pdb).
